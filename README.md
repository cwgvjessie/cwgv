# 圖片縮放工具

一款純前端圖片縮放工具，全程在瀏覽器本機執行，圖片不上傳至伺服器。

## 功能

- **純計算模式**：手動輸入原始尺寸，即時計算等比例縮放後的目標尺寸
- **圖片縮放模式**：上傳圖片後，依設定尺寸縮放並下載
- 寬度、高度、縮放比例（%）三欄即時聯動，永遠維持等比例
- 支援輸出格式：JPG / PNG / WebP
- 支援 JPG / WebP 品質調整（1–100）

## 支援格式

JPG、PNG、WebP、GIF（靜態）、BMP，檔案大小上限 20MB

## 使用方式

直接開啟 [index.html](index.html) 或前往 GitHub Pages 網址即可使用，無需安裝任何套件。

## 開發

單一 `index.html`，無建置步驟，無外部依賴。
