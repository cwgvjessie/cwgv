# 會員中心 - 更換信用卡功能規格書

## 1. 功能概述
本功能允許已開啟「自動續訂」的會員，在不取消訂閱的情況下，更換綁定的信用卡。這有助於解決信用卡過期、遺失或額度不足導致扣款失敗的問題。

## 2. 用戶操作流程 (User Flow)
1. **進入頁面**：用戶登入會員中心，進入「訂戶管理」頁面。
2. **點擊變更**：在自動續訂的方案資訊中，點擊信用卡末四碼旁邊的「變更」按鈕。
3. **跳轉金流**：系統引導用戶至第三方金流平台（如藍新科技）進行信用卡資訊輸入與驗證（通常會進行 $1 元或免額核授）。
4. **完成返回**：金流驗證完成後，自動跳回「訂戶管理」頁面。
5. **更新狀態**：頁面顯示「更新成功」提示，並立即顯示新的信用卡末四碼。

## 3. 介面需求 (UI/UX)
- **觸發按鈕**：應在「扣款方式：信用卡 末四碼 XXXX」旁提供明顯的「變更」連結或按鈕。
- **過渡提示**：在跳轉至金流頁面前，可顯示「正在前往安全支付頁面...」的載入狀態。
- **結果通知**：
    - **成功**：顯示綠色 Toast 或訊息盒「✅ 信用卡更換成功」。
    - **失敗**：顯示紅色 Toast「❌ 更換失敗：請洽金流服務商或重新嘗試」。

## 4. 例外處理
- **金流端取消**：若用戶在金流頁面點擊取消返回，應回到原頁面並顯示「已取消變更操作」。
- **授權失敗**：若新卡授權失敗，應維持舊卡資訊，並提示用戶更換其他卡片。
